Computer Vision para trailers de supermercado

I led a team in the development of a Computer Vision project for detecting objects and empty spaces in supermarket trailers, using YOLOv8 for object detection. The project includes a Streamlit interface, a FastAPI backend, and Redis to manage tasks efficiently. 🚀
